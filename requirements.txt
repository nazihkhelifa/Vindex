# Python dependencies for the Vindex / A-Vindex toolkit

# Inference is **vindex-only**: nothing here loads a Hugging Face model
# for prediction. The `transformers` / `accelerate` / `sentencepiece`
# packages are intentionally absent — they were only used by the old
# HF inference fallbacks, which have been removed.
#

#
# Two install profiles:
#
#   1. Minimum (run an existing vindex — no extraction):
#          pip install numpy tokenizers
#      Enough for vindex-infer.py, vindex-infernce-next.py,
#      avindex-infer.py (probe/walk/scan), avindex-infer-next.py.
#
#   2. Full (also lets you build vindexes from a HF model):
#          pip install -r requirements.txt
# ──────────────────────────────────────────────────────────────────────
